## Technologies used
- #### HTML
- #### CSS
- #### JS (vanilla)

## Features

- Play and Pause song
- Navigate to Next and Prev songs
- Added a small black stick for old media player effect
- Seek through the song by clicking the progressbar
- Added a duration tracker
